# Parametre.ini
```
[Titre]
Name=Your Title

[Dossiers]
C:\Your path

[Infos]
Name=oui
Date=oui
Taille=oui

[Extensions]
.txt
.jpg
.png

[Exclusions]
All folder you don't want put the path
```
